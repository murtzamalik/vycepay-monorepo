28 screens · Dark theme · HTML-to-Figma ready · March 2026

How to import into Figma

  1. Install "HTML to Figma" plugin by builder.io in Figma (or use the Chrome extension)
  2. Open each screen HTML file in Chrome
  3. In Figma, run the HTML to Figma plugin
  4. Click "Import from browser tab" — it captures the active Chrome tab
  5. Each screen imports as a Figma frame at 390px width
  6. Repeat for all 28 screens
